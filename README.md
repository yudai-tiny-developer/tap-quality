# YouTube Playback Quality Control

A Chrome extension that adds one-tap playback quality buttons directly to the YouTube player.

It is designed for touch-friendly use, but it also supports keyboard shortcuts so you can switch quality levels without opening YouTube's settings menu.

## Features

- Adds quality buttons to the YouTube player controls
- Supports 144p, 240p, 360p, 480p, 720p, 1080p, 1440p, 2160p, and Auto
- Lets you enable or disable each button from the extension popup
- Supports keyboard shortcuts through Chrome extension commands

## Usage

1. Open any video on `https://www.youtube.com/`.
2. Look for the added quality buttons in the player controls.
3. Click a quality button to switch playback quality.
4. Open the extension popup to show or hide specific buttons.

## Keyboard Shortcuts

You can customize shortcuts in Chrome at `chrome://extensions/shortcuts`.
